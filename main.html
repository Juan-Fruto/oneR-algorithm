Desarrollo


1.- Ingresa al entorno de desarrollo de R.

2.- Ahora debes indicar a R que usars el paquete OneR para poder generar modelos de reglas de inferencia, para eso teclea:

install.packages(OneR, repos=http://cran.us.r-project.org) Una vez que instalas ese paquete, debes indicar que, de todas las bibliotecas que contiene el paquete OneR, utilizars alguna de ellas. Como en esta prctica usars la biblioteca OneR, entonces debes teclear. 
library("OneR")
library("readxl")

3. Para realizar este ejemplo, utilizaremos un data set llamado Iris. Este data set quiz es uno de los ms conocidos para analizar algoritmos de reconocimiento de patrones. Contiene tres clases de 50 instancias cada una, cada clase se refiere a una planta de iris [1]. La informacin de los atributos de ese data set es [3]:

  1. sepal length in cm
  2. sepal width in cm
  3. petal length in cm
  4. petal width in cm
  5. class: Iris Setosa Iris Versicolour Iris Virginica

    Al instalar la biblioteca OneR tienes acceso al data set Iris. Para abrirlo usa la funcin optbin de la siguiente forma: 

datos <- optbin(iris)

4. Con la funcin anterior ya tienes en memoria el data set Iris, el cual se accede mediante la lista datos. Ahora debes crear el modelo de reglas, as que debes usar la funcin OneR. 

modelo <- OneR(datos, verbose = TRUE)

Tambin pude leer en dicha documentacin que este algoritmo no logra generalizar bien las reglas (overfiting) cuando hay columnas con una gran variedad de valores categricos, como nombres.
